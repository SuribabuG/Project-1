ReactJS
========
It is a declarative, efficient and flexible javascript frontend library used to develop frontend applications and user interfaces.

It is an open source, component based frontend javascript library responsible only for view layer of the application.

It is developed by Jordan Walke who was software engineer at Facebook.

It is initially developed by Facebook and  it is used in their own products like Whatsapp and Instagram.

It was released to public in the Month of May, 2013.

The latest version of reactjs is v18.x.

Official website for reactjs is i.e https://react.dev/ 

It is used to develop reusable components.

A component is a building block of react application.
